# Core

## Основные функции
- Создание/изменение заметок
- Теги для заметок
- Объедение заметок в группу
- Синхронизация между устройствами
- Уведомления по заметкам
- Задачи (сбор по заметкам в одно место с заголовками)
- Календарь

***

### Скелет заметки
- Автор заметки
- Время создания заметки
- Приоритет заметки
- Вложения заметки
- Текст заметки
- Тег заметки
- Группа заметки
