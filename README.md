# Основной концепт
1. Бэкенд с логикой 
    - Шифрование
      - ChaCha20+Poly1305 - шифрование
      - HMAC256 - хеширование
    - Бизнес логика(1 модуль = 1 язык)
      - Rust
      - C/C++
      - Python
      - Go
      - Многопоток
2. АПИшка
	- RESTApi
3. База данных
	- Реляционная
		- PostrgreSQL
	- Не Реляционная
		- MongoDB
	- Работать с кешем
		- Redis
4. Логирование
	- Метрики
		- Prometheus + grafana
	- Логи
		- Vector + Loki
5. Фронтенд приложения
	- Фреймворк
		- VUE.js
6. Масштабирование
	- Docker
	- k8s + ansible
	- terraform
7. Балансировка нагрузки
	- Nginx
8. Систему работы с очередью
	- RabbitMQ
